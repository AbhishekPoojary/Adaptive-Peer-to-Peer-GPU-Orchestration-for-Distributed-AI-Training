🖥️ Share Your GPU with the Cluster
You can contribute your GPU (or CPU) to the training cluster. The orchestrator will automatically schedule jobs to your node based on load and reliability. Your machine runs a lightweight agent that only accepts commands from the orchestrator.

How to join in 4 steps:

1
Install the agent on your machine:
pip install fastapi uvicorn httpx psutil torch pydantic-settings python-dotenv
2
Set environment variables (replace values for your machine):
set NODE_ID=my-gpu-node set AGENT_HOST=YOUR_IP_HERE set AGENT_PORT=8001 set ORCHESTRATOR_URL=http://localhost:8000 set ORCHESTRATOR_API_KEY=your-api-key
3
Start the agent:
uvicorn gpu_agent.agent:app --host 0.0.0.0 --port 8001
4
Click Register My GPU above to manually register, or your agent registers automatically on startup.
